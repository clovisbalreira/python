# Alura - Imersão em Dados

📌 **Descrição**
Cursos de Imersão em Dados da Alura com análise exploratória, visualizações e projetos práticos.

---

## 🚀 Tecnologias
- Python 3.7+
- Pandas
- Plotly
- Jupyter Notebooks
- Flask

---

## 🎯 Funcionalidades
- Análise exploratória de dados
- Dashboards interativos
- Visualizações de dados
- Notebooks educacionais

### Versões
- **Imersão 2025** - Primeira edição do curso
- **Imersão 2026** - Versão atualizada com melhorias

---

## ▶️ Como rodar

1. Para Jupyter Notebooks:
```bash
cd alura/imersao-dados-2025
jupyter notebook main.ipynb
```

2. Para ao Dashboard:
```bash
cd alura/imersao-dados-2025/dashboard
pip install -r requirements.txt
python app.py
```
