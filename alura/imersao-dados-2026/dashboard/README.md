# Imersão em Dados 2026 - Dashboard

📌 **Descrição**
Dashboard atualizado da Imersão em Dados da Alura 2026, com análise avançada de dados e visualizações aprimoradas em Python.

---

## 🚀 Tecnologias
- Python 3.8+
- Flask / Streamlit
- Pandas
- Plotly
- Dataset Alura 2026

---

## 🎯 Funcionalidades
- Dashboard web interativo
- Análise exploratória completa
- Visualizações avançadas
- Processamento de dados em tempo real

---

## ▶️ Como rodar

1. Navegue até a pasta do projeto:
```bash
cd alura/imersao-dados-2026/dashboard
```

2. Instale as dependências:
```bash
pip install -r requirements.txt
```

3. Execute a aplicação:
```bash
python app.py
```

4. Acesse no navegador:
```
http://localhost:5000
```
