## ▶️ Como Rodar

### Pré-requisitos
- Python 3.7+
- pip (gerenciador de pacotes)

### Executar um arquivo Python simples
```bash
python arquivo.py
```

### Para Jupyter Notebooks
```bash
jupyter notebook arquivo.ipynb
```

### Para as aplicações Web (Flask/Django)
```bash
# Instalar dependências
pip install -r requirements.txt

# Executar Flask
python app.py

# Executar Django
python manage.py runserver
```

### Para projetos de Ciência de Dados
```bash
pip install -r requirements.txt
jupyter notebook
```
