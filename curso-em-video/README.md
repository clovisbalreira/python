# Curso em Vídeo - Python

📌 **Descrição**
Série completa de módulos do Curso em Vídeo de Python, com desafios progressivos e exercícios de lógica de programação.

---

## 🚀 Tecnologias
- Python 3.7+
- Lógica de Programação
- Estruturas de Dados

---

## 🎯 Funcionalidades
- 20+ desafios progressivos por módulo
- Exercícios de lógica
- Conceitos fundamentais
- Progressão de dificuldade

### Módulos
- **Módulo 01** - Fundamentos (variáveis, tipos de dados, operadores)
- **Módulo 02** - Estruturas de controle (if, while, for)
- **Módulo 03** - Funções e modularização
- **Módulo 04** - Estruturas de dados e bibliotecas

---

## ▶️ Como rodar

1. Navegue para um desafio específico:
```bash
cd curso-em-video/modulo-01/desafios
```

2. Execute o desafio:
```bash
python desafio-001.py
```

3. Para resolver os exercícios:
```bash
cd ../exercicios
```
