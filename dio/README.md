# DIO - REST e SOAP

📌 **Descrição**
Projetos da plataforma DIO com estudos de arquitetura REST e SOAP, comunicação entre APIs e integração de serviços web.

---

## 🚀 Tecnologias
- Python 3.7+
- Requests
- Zeep (SOAP)
- JSON / XML
- Jupyter Notebooks

---

## 🎯 Funcionalidades
- Consumo de APIs REST
- Integration com serviços SOAP
- Requisições HTTP
- Tratamento de dados JSON/XML
- Notebooks interativos

### Projetos
- **REST** - Integração com APIs RESTful
- **SOAP** - Integração com serviços SOAP

---

## ▶️ Como rodar

1. REST:
```bash
cd dio/rest
jupyter notebook conexaoRest.ipynb
```

2. SOAP:
```bash
cd dio/soap
jupyter notebook conexaoSoap.ipynb
```

3. Instale as dependências:
```bash
pip install requests zeep
```
