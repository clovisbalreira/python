# Flappy Bird - Jogo em Python

📌 **Descrição**
Recriação do famoso jogo Flappy Bird usando Python com Pygame, incluindo mecânicas clássicas e desafios.

---

## 🚀 Tecnologias
- Python 3.7+
- Pygame
- POO (Programação Orientada a Objetos)

---

## 🎯 Funcionalidades
- Mecânica de voo de pássaro
- Obstáculos dinâmicos
- Sistema de pontuação
- Colisão e física do jogo

---

## ▶️ Como rodar

1. Navegue até a pasta do projeto:
```bash
cd hashtag/flappy-bird
```

2. Instale o Pygame:
```bash
pip install pygame
```

3. Execute o jogo:
```bash
python FlappyBird.py
```

4. Controles:
- **ESPAÇO** - Fazer o pássaro voar
- **ESC** - Sair do jogo

---

## 📚 Aprendizados
- Desenvolvimento de jogos com Pygame
- Programação orientada a objetos
- Renderização de gráficos
- Detecção de colisões
- Loop de game engine
