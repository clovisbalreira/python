# Minicurso Hashtag - Ciência de Dados

📌 **Descrição**
Minicurso completo de Ciência de Dados da Hashtag com análise exploratória, tratamento de dados, machine learning e deploy de modelos.

---

## 🚀 Tecnologias
- Python 3.7+
- Pandas
- Scikit-Learn
- Matplotlib / Seaborn
- Jupyter Notebooks

---

## 🎯 Funcionalidades
- Análise exploratória de dados (EDA)
- Tratamento e limpeza de dados
- Aprendizado de máquina para classificação
- Visualizações avançadas
- Deploy de modelos

### Conteúdo
- **Aula 2** - Conceitos fundamentais do Pandas
- **Aula 3** - Análise Exploratória
- **Aula 4** - Machine Learning para Classificação
- **Aula 5** - Tratamento de Dados
- **Aula 6** - Deploy de Modelos

---

## ▶️ Como rodar

1. Navegue até a pasta:
```bash
cd hashtag/minicurso-hashtag-ciencia-de-dados
```

2. Instale as dependências:
```bash
pip install pandas scikit-learn matplotlib seaborn jupyter
```

3. Abra os notebooks:
```bash
jupyter notebook initial.ipynb
```
