# Mudando Nome de Arquivos - Automação

📌 **Descrição**
Script Python para renomear arquivos em massa de forma automatizada e programática.

---

## 🚀 Tecnologias
- Python 3.7+
- OS / Pathlib

---

## 🎯 Funcionalidades
- Renomeação em lote
- Padrões de renomeação customizáveis
- Validação de nomes
- Segurança contra sobrescrita

---

## ▶️ Como rodar

1. Navegue até a pasta:
```bash
cd hashtag/mudando-nome-arquivos
```

2. Execute o script:
```bash
python initial.py
```

---

## 📚 Aprendizados
- Manipulação de sistema de arquivos
- Automação using Python
- Strings e padrões
