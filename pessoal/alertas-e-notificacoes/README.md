# Alertas e Notificações

📌 **Descrição**
Script Python para criar e gerenciar sistema de alertas e notificações.

---

## 🚀 Tecnologias
- Python 3.7+
- Bibliotecas de notificação
- Sistema de eventos

---

## 🎯 Funcionalidades
- Envio de alertas personalizados
- Gerenciamento de notificações
- Sistema de eventos em tempo real

---

## ▶️ Como rodar

1. Navegue até a pasta do projeto:
```bash
cd alertas-e-notificacoes
```

2. Execute o script:
```bash
python initicial.py
```

---

## 📚 Aprendizados
- Implementação de sistema de alertas
- Gerenciamento de eventos
- Sincronia entre processos
