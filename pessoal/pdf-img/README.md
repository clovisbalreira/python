# PDF e Imagens - Conversão e Manipulação

📌 **Descrição**
Script Python para converter, manipular e processar arquivos PDF e imagens usando bibliotecas especializadas.

---

## 🚀 Tecnologias
- Python 3.7+
- Pillow (PIL)
- PyPDF2
- PDF2Image

---

## 🎯 Funcionalidades
- Conversão PDF para imagem
- Conversão imagem para PDF
- Manipulação de imagens (resize, crop, etc)
- Processamento em lote
- Extração de páginas

---

## ▶️ Como rodar

1. Navegue até a pasta do projeto:
```bash
cd pessoal/pdf-img
```

2. Instale as dependências:
```bash
pip install Pillow PyPDF2 pdf2image
```

3. Execute o script:
```bash
python main.py
```

## 📚 Aprendizados
- Manipulação de imagens com Pillow
- Processamento de PDFs
- Conversão entre formatos
- Processamento em lote de arquivos
