# Criar PDF - Gerador de Documentos

📌 **Descrição**
Script Python para gerar, manipular e criar arquivos PDF de forma automatizada e programática.

---

## 🚀 Tecnologias
- Python 3.7+
- ReportLab / PyPDF2
- FPDF

---

## 🎯 Funcionalidades
- Geração de PDFs dinâmicos
- Fusão de múltiplos PDFs
- Extração de conteúdo PDF
- Importação de templates
- Preenchimento automático

---

## ▶️ Como rodar

1. Navegue até a pasta do projeto:
```bash
cd pessoal/criar-pdf
```

2. Instale as dependências:
```bash
pip install reportlab pypdf2
```

3. Execute o script:
```bash
python initial.py
```

4. Os PDFs gerados estarão na pasta `arquivos/`
