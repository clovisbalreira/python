## 🎯 Funcionalidades
- Técnicas avançadas de Python
- Otimização de código

- Notebooks educacionais

---

## ▶️ Como rodar

1. Navegue até a pasta:
```bash
cd pessoal/python-powerup
```

2. Instale as dependências:
```bash
pip install pandas numpy jupyter
```

3. Abra o notebook:
```bash
jupyter notebook inicial.ipynb
```
