# Separando Imagem do Fundo Branco

📌 **Descrição**
Script Python para remover e separar imagens do fundo branco, útil para processamento de fotos e imagens.

---

## 🚀 Tecnologias
- Python 3.7+
- Pillow (PIL)
- OpenCV

---

## 🎯 Funcionalidades
- Remoção automática de fundo branco
- Processamento de imagens
- Separação de objetos
- Exportação com transparência

---

## ▶️ Como rodar

1. Navegue até a pasta:
```bash
cd pessoal/separando_imagem_fundo_branco
```

2. Instale as dependências:
```bash
pip install Pillow opencv-python
```

3. Execute o script:
```bash
python index.py
```

- Processamento de imagens
- Manipulação de cores
- Remoção de fundos
