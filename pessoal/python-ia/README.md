Projetos e estudos em Python focados em Inteligência Artificial, Machine Learning e análise de dados com bibliotecas especializadas.

- Scikit-Learn

- NumPy
- Jupyter Notebooks

---

## 🎯 Funcionalidades
- Modelos de machine learning
- Análise de dados e visualizações
- Tratamento de datasets
- Treinamento e avaliação de modelos
- Notebooks interativos

---

## ▶️ Como rodar

1. Navegue até a pasta do projeto:
```bash
cd pessoal/python-ia
```

2. Instale as dependências:
```bash
pip install -r requirements.txt
```

3. Abra os notebooks:
```bash
jupyter notebook inicial.ipynb
```

4. Execute os scripts Python:
```bash
python script.py
```
