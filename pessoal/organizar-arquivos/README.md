# Organizar Arquivos - Automação de Sistema

📌 **Descrição**
Script Python para automatizar a organização e gerenciamento de arquivos em diretórios do sistema.

---

## 🚀 Tecnologias
- Python 3.7+
- OS / Shutil
- Path (pathlib)

---

## 🎯 Funcionalidades
- Organização automática de arquivos por tipo
- Renomeação e movimentação em massa
- Limpeza de diretórios
- Categorização por extensão

---

## ▶️ Como rodar

1. Navegue até a pasta do projeto:
```bash
cd pessoal/organizar-arquivos
```

2. Execute o script:
```bash
python initial.py
```

3. Os arquivos na pasta `arquivos/` serão organizados automaticamente

---

## 📚 Aprendizados
- Manipulação do sistema de arquivos
- Automatização de tarefas repetitivas
- Trabalho com paths e diretórios
- Script de utilidade do sistema
