# Tabela - Interface de Dados

📌 **Descrição**
Projeto de criação e manipulação de tabelas em Python usando interface gráfica (GUI) com libraries como tkinter ou similar.

---

## 🚀 Tecnologias
- Python 3.7+
- Tkinter / PySimpleGUI
- Pandas (opcional)

---

## 🎯 Funcionalidades
- Exibição de tabelas em GUI
- Manipulação de dados visual
- Interface interativa
- Radio buttons e controles

---

## ▶️ Como rodar

1. Navegue até a pasta:
```bash
cd pessoal/tabela
```

2. Execute os scripts:
```bash
python tabela.py
python radio.py
```

- Criação de interfaces gráficas
- Manipulação de widgets
- Exibição de dados em GUI
- Interação visual com dados
